# Proyectos de Data Science — Constanza Belén Lobo

Repositorio con proyectos desarrollados durante la carrera de Data Science (Coderhouse).

## Proyectos

- [Producción Agrícola y Ganadera + Clima](./proyecto-agro-clima/) — Análisis de estructuras productivas globales (1961-2023) e integración con datos climáticos para modelado predictivo.
- [NLP - Reseñas de E-Commerce](./proyecto-nlp/) — Análisis de sentimiento y topic modeling sobre reseñas de un e-commerce de ropa femenina, con modelos clásicos (TF-IDF + Regresión Logística) y red neuronal en PyTorch.
